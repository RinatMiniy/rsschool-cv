Code example

Input - number

Output - table of size n*n filled with numbers from 1 to n**2 in a spiral starting from the upper left corner and twisted clockwise

n = int(input())
m = [[0] * n for i in range(n)]
i, j, di, dj = 0, 0, 0, 1
for k in range(n * n):
m[i][j] = k + 1
if (not -1 < i + di < n) or (not -1 < j + dj < n)
or m[i + di][j + dj] != 0:
di, dj = dj, -di
i, j = i + di, j + dj
[print(*i) for i in m]
